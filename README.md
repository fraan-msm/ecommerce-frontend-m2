\# Ecommerce Frontend – Módulo 2



Proyecto de frontend desarrollado como parte del Módulo 2 del curso, consistente en un sitio web de tipo e-commerce para un emprendimiento de productos de pastelería tradicional y vegana llamado \*\*El Hornito de la Pan\*\*.



\## Descripción del proyecto



El proyecto simula una tienda online que permite:

\- Visualizar un listado de productos

\- Acceder al detalle de un producto

\- Simular agregar productos a un carrito de compras



Está desarrollado utilizando HTML, CSS, JavaScript y Bootstrap.



\## Tecnologías utilizadas



\- HTML5

\- CSS3

\- JavaScript

\- Bootstrap 5

\- Git y GitHub



\## Estructura del proyecto



\- `index.html`: página principal con el listado de productos

\- `producto-detalle.html`: vista de detalle de un producto

\- `carrito.html`: página del carrito de compras

\- `assets/`: carpeta de recursos (CSS, imágenes y JavaScript)
